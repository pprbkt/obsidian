## Overview

This repository hosts my personal [Obsidian](https://obsidian.md/) vault — a continuously growing collection of notes, study material, research, and documentation accumulated over time. It serves as a single, organized source of reference for everything I learn and work on.

---

## Setup

**Clone the vault**

```bash
git clone https://github.com/paperbukit/obsidian.git
```

**Open in Obsidian**

1. Launch Obsidian
2. Select **Open folder as vault**
3. Point it to the cloned directory
4. Install any recommended community plugins from `.obsidian/plugins/`

> Some internal links may appear broken until Obsidian finishes indexing the vault — this resolves on its own.

---

## Why Obsidian

- **Local-first** — notes are stored as plain files, not locked in a proprietary cloud
- **Plain Markdown** — future-proof, portable, and version-controllable via Git
- **Bi-directional linking** — build a connected graph of knowledge over time
- **Extensible** — a rich plugin ecosystem to fit any workflow
- **Fast** — performs well even at scale

---

## Notes

This vault is personal and opinionated in structure. Notes are written for my own understanding first — clarity and recall over presentation. Content is added and revised continuously.
